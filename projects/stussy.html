Stussy

A powerful and user-friendly static site generator.

Convert Markdown files into beautiful websites with advanced features like themes, tags, categories, social links, RSS feed, and SEO optimization.
Perfect for blogs, documentation, and personal websites.

Core Features

Everything you need to create a professional website from Markdown files.

  • Markdown Writing Write content in simple Markdown format
  • Static Pages Create About, Contact, and other static pages
  • Tags & Categories Organize content with flexible taxonomy
  • Custom Templates Fully customizable HTML templates
  • Fast & Lightweight Optimized for speed and performance
  • Easy to Use Beginner-friendly with clear documentation

Advanced Features

Professional features for modern websites.

  • Social Links GitHub, Twitter, LinkedIn, and more
  • RSS Feed Automatic RSS feed generation
  • SEO Optimized Open Graph and Twitter Cards support
  • Featured Images Support for post and page images
  • Blog Archive Organized post history and navigation
  • Responsive Design Mobile-friendly out of the box
